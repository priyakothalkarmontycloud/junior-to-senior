# Mission Established: Strategic Programming in the AI Era

Priya is an experienced developer who has identified that AI tools are rapidly automating
tactical coding, making strategic skills the new table stakes. She has clear conviction about
the direction — no need to motivate the "why." She works in a cloud management SaaS context
(MontyCloud), which provides a concrete grounding for all future lessons.

**Implications:** Start with immediately actionable skills (scoping, ADRs, system design) rather
than theory. Use cloud/SaaS examples. Assume solid coding fundamentals — do not teach
programming basics.
